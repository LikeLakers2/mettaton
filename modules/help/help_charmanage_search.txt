```rp!charmanage search```
**Usage**:
rp!charmanage search `<field/property>` `<operator>` `<search text>` `[page]`

**Parameters**:
`1.` The field or property name to search. Case-insensitive.
`2.` The operator to search using. Execute `rp!help charmanage search op` for help with this.
`3.` The search text to use for this search. Case-insensitive.
`4.` The page of search results to return.

**Description**:
Searches the character list for this server, for any characters that match the specified search terms. Useful if you swear you remember a character but can't quite get it off the tip of your tounge.
